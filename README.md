# 🛒 E-Commerce SQL Data Analysis Project

## 📌 Overview
This project focuses on analyzing an e-commerce database using SQL to extract meaningful business insights.  
It simulates real-world scenarios like sales tracking, customer behavior analysis, and product performance.

---

## 🎯 Objectives
- Analyze overall business revenue
- Identify top-performing products
- Understand customer purchasing patterns
- Track monthly sales trends

---

## 🧰 Tech Stack
- MySQL
- SQL

---

## 🗂️ Database Structure
The project includes multiple relational tables such as:
- Customers
- Orders
- Products
- Order Details

📌 Relationships are designed using proper primary and foreign keys.

---

## 📊 Key Analysis Performed
✔ Total Revenue Calculation  
✔ Top-Selling Products  
✔ Customer Purchase Insights  
✔ Monthly Sales Trends  
✔ Order Distribution Analysis  

---

## 🧠 SQL Concepts Used
- Joins (INNER JOIN, LEFT JOIN)
- GROUP BY & Aggregations
- Subqueries
- Window Functions (RANK, ROW_NUMBER)
- Filtering & Sorting
